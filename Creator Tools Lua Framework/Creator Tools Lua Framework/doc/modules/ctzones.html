Module ctzones

Creator Tools LUA Zone Utilities File

Author: Native Instruments Written by: Yaron Eshkar

This file includes useful functions for usage in Creator Tools Lua scripts.

Simply include this line in any script (if running a script from another location that users this file, make sure to point to the correct path): local ctZones = require("CtZones")

Then you can simply call any function like: ctZones.test_function()

It is also possible of course to copy entire specific functions from this list directly into your script. In that case remove the ctZones part from the function name, and then simply call it normally like: test_function()

CtZones.zone_param_single (source_group, source_zone, dest_group, dest_zone, param)
Copy zone parameters from one zone to another.

Parameters:

  • source_group int the group index of the source group.
  • source_zone int the zone index of the source zone within the source group.
  • dest_group int the group index of the destination group.
  • dest_zone int the zone index of the destination zone within the destination group.
  • param string if set to "all" copies all zone parameters.

Returns:

    bool
CtZones.zone_root (start_root, start_zone, until_zone, start_group, until_group, increment, confine)
Set the root note of a range of zones over a range of groups with optional arguments.

Parameters:

  • start_root int the root note number to start from for every zone. Defaults to 60.
  • start_zone int the index of the zone to start the iteration from. Defaults to 0.
  • until_zone int the index of the zone to end the iteration on. Defaults to 0.
  • start_group int the index of the group to start the iteration from. Defaults to 0.
  • until_group int the index of the group to end the iteration on. Defaults to 0.
  • increment bool when true will increment the root note for each zone in the set zone range starting from the start_root value.
  • confine bool when true, sets the zone low key and high key to the same value as the root note.

Returns:

    bool
CtZones.zones_with_same_key (root_key, group)
Find and count the number of all zones that have the same root key within a group or all groups.

Parameters:

  • root_key int the root key to compare.
  • group int the group to check in, if none specified checks in all groups.

Returns:

    table returns a table containing a list of zones that are in the same key as the specified one.
