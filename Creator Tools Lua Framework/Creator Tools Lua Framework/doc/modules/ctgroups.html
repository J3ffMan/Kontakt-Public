Module ctgroups

Creator Tools LUA Group Utilities File

Author: Native Instruments Written by: Yaron Eshkar

This file includes useful functions for usage in Creator Tools Lua scripts.

Simply include this line in any script (if running a script from another location that users this file, make sure to point to the correct path): local ctGroups = require("CtGroups")

Then you can simply call any function like: ctGroups.test_function() It is also possible of course to copy entire specific functions from this list directly into your script. In that case remove the CtGroups part from the function name, and then simply call it normally like: test_function()

Functions

CtGroups.test_function ()
Test Function. Takes no arguments, prints to console when called.
CtGroups.group_count ()
Check group count. Takes no arguments, check and print the total group count of the instrument.
CtGroups.group_create (number, insert_at, group_name, increment, increment_start)
Create an x number of groups. Optionally choose where to insert and if and how to rename.

Parameters:

  • number int the number of groups to create.
  • insert_at int the group index after which to start creating new groups.
  • group_name string the name for the created groups.
  • increment bool when set to true, add an incremental numeric count to the group name.
  • increment_start int the number from which to start incrementing from.

Returns:

    bool
CtGroups.group_copy (group_to_copy, times_to_paste, insert_at, group_name, increment, increment_start)
Copy a group and paste it x times. Optionally at the end or by insertion as well as optionally rename the copied groups.

Parameters:

  • group_to_copy int the group to copy.
  • times_to_paste int how many copies should be created.
  • insert_at int the group index after which to start creating new groups.
  • group_name string the name for the created groups.
  • increment bool when set to true, add an incremental numeric count to the group name.
  • increment_start int the number from which to start incrementing from.

Returns:

    bool
CtGroups.groupInfo (group)
Print detailed information about a group or all groups.

Parameters:

  • group int the group to check. If no group is given, report on all the groups.

Returns:

    bool
CtGroups.group_name_replace (replace_this, replace_with, start_group, until_group)
Replace group name. Searches for a string in the group name and replaces it with a new string.

Parameters:

  • replace_this string the sub-string in the group name to look for.
  • replace_with string the sub-string to replace matches with.
  • start_group int the index of the first group to start checking in. Defaults to 0.
  • until_group int the index of the last group to start checking in. Defaults to start_group.

Returns:

    bool
CtGroups.group_name_search (search_name, start_group, until_group, contains)
Search for a group name or a string in the name. Return a table with all groups that match.

Parameters:

  • search_name string the group name or sub-string in the name to look for.
  • start_group int the index of the first group to start checking in. Defaults to 0.
  • until_group int the index of the last group to start checking in. Defaults to start_group.
  • contains bool when true searches for a sub-string within the group name, otherwise searches for matches in the entire group name.

Returns:

    table returns a table with the indexes of the groups matching the condition.
CtGroups.group_remove (start_group, until_group)
Remove groups within a range of groups.

Parameters:

  • start_group int the index of the first group to remove. Defaults to 0.
  • until_group int the index of the last group to remove. Defaults to start_group.

Returns:

    bool
CtGroups.group_rename (new_name, start_group, until_group, increment, increment_start)
Rename groups. Change name and an add optional incremental count

Parameters:

  • new_name string the new name for the groups.
  • start_group int the index of the group to start renaming. Defaults to 0.
  • until_group int the index of the last group to be renamed. Defaults to start_group.
  • increment bool when true will add an incremental count to every group name.
  • increment_start int the number from which to start the incremental count from. Defaults to 1.

Returns:

    bool
CtGroups.group_names ()
Print group names. Takes no arguments, prints all group names.

Returns:

    bool
CtGroups.group_params (group, volume, tune, pan, name)
Set all group parameters.

Parameters:

  • group int the index of the group to set. Defaults to 0.
  • volume int the group volume. Defaults to 0.
  • tune int the group tune. Defaults to 0.
  • pan int the group pan. Defaults to 0.
  • name string the name of the group, if specified.

Returns:

    bool
